canvas

微信小程序支付宝小程序
docdoc
差异属性说明是否支持返回值差异备注
type指定 canvas 类型，当前仅支持 webgl不支持
canvas-idcanvas 组件的唯一标识符，若指定了 type 则无需再指定该属性完全支持
disable-scroll当在 canvas 中移动时且有绑定手势事件时，禁止屏幕滚动以及下拉刷新完全支持
bindtouchstart手指触摸动作开始完全支持
bindtouchmove手指触摸后移动完全支持
bindtouchend手指触摸动作结束完全支持
bindtouchcancel手指触摸动作被打断，如来电提醒，弹窗完全支持
bindlongtap手指长按 500ms 之后触发，触发了长按事件后进行移动不会触发屏幕的滚动完全支持
binderror当发生错误时触发 error 事件不支持
