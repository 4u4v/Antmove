icon

微信小程序支付宝小程序
docdoc
差异属性说明是否支持返回值差异备注
aria-label无障碍访问，（属性）元素的额外描述不支持
sizeicon的大小，单位px支持支付宝小程序只支持数值类型
coloricon的颜色，同css的color完全支持
typeicon的类型，有效值：success, success_no_circle, info, warn, waiting, cancel, download, search, clear完全支持

progress

微信小程序支付宝小程序
docdoc
差异属性说明是否支持返回值差异备注
percent百分比0~100完全支持
show-info在进度条右侧显示百分比完全支持
border-radius圆角大小不支持
font-size右侧百分比字体大小不支持
stroke-width进度条线的宽度支持支付宝小程序只支持数值类型
color进度条颜色完全支持
activeColor已选择的进度条的颜色完全支持
backgroundColor未选择的进度条的颜色完全支持
active进度条从左往右的动画完全支持
active-modebackwards: 动画从头播；forwards：动画从上次结束点接着播不支持
bindactiveend动画完成事件不支持
aria-label无障碍访问，（属性）元素的额外描述不支持
