openLocation

函数名说明微信小程序支付宝小程序是否支持
openLocation打开位置docdoc完全支持

getLocation

函数名说明微信小程序支付宝小程序是否支持
getLocation获取位置docdoc支持

  • 参数缺失, 参数type的取值/默认值差异, 返回值缺失/差异
差异参数说明差异类型
altitude传入 true 会返回高度信息，由于获取高度需要较高精确度，会减慢接口返回速度不支持该属性
type的取值wx: 默认值wgs84, alipay: 默认值0默认值不同
差异属性说明差异类型
speed速度，单位 m/s不支持该属性
altitude高度，单位 m不支持该属性
verticalAccuracy垂直精度，单位 m（Android 无法获取，返回 0）不支持该属性
error返回值差异app未授权定位：alipay返回错误码11, wx返回errMsg：'getLocation:fail 1', app授权，小程序弹框询问权限时拒绝：alipay返回错误码2001；wx返回errMsg: “getLocation:fail authorize no response”类型不同

chooseLocation

函数名说明微信小程序支付宝小程序是否支持
chooseLocation打开地图选择位置docdoc完全支持
