movable-view

属性名说明是否支持备注
directionmovable-view的移动方向，属性值有all、vertical、horizontal、none完全支持支付宝小程序完全支持该属性
inertiamovable-view是否带有惯性不支持暂不支持
out-of-boundsmovable-view是否带有惯性不支持暂不支持
x定义x轴方向的偏移，如果x的值不在可移动范围内，会自动移动到可移动范围；改变x的值会触发动画完全支持支付宝小程序完全支持该属性
y定义y轴方向的偏移，如果y的值不在可移动范围内，会自动移动到可移动范围；改变y的值会触发动画完全支持支付宝小程序完全支持该属性
damping阻尼系数，用于控制x或y改变时的动画和过界回弹的动画，值越大移动越快不支持暂不支持
friction阻尼系数，用于控制x或y改变时的动画和过界回弹的动画，值越大移动越快不支持暂不支持
disabled是否禁用完全支持支付宝小程序完全支持该属性
scale是否支持双指缩放，默认缩放手势生效区域是在movable-view内不支持暂不支持
scale-min定义缩放倍数最小值不支持暂不支持
scale-max定义缩放倍数最大值不支持暂不支持
scale-value定义缩放倍数，取值范围为 0.5 - 10不支持暂不支持
animation是否使用动画不支持暂不支持
bindchange拖动过程中触发的事件，event.detail = {x, y, source}完全支持完全支持
bindscale缩放过程中触发的事件，event.detail = {x, y, scale}，x和y字段在2.1.0之后支持不支持暂不支持
htouchmove初次手指触摸后移动为横向的移动时触发，如果catch此事件，则意味着touchmove事件也被catch不支持暂不支持
vtouchmove初次手指触摸后移动为纵向的移动时触发，如果catch此事件，则意味着touchmove事件也被catch不支持暂不支持

scroll-view

属性名说明是否支持备注
scroll-x允许横向滚动完全支持支付宝小程序完全支持该属性
scroll-y允许纵向滚动完全支持支付宝小程序完全支持该属性
upper-threshold距顶部/左边多远时，触发 scrolltoupper 事件支持支付宝小程序只支持number类型
lower-threshold距顶部/左边多远时，触发 scrolltoupper 事件支持支付宝小程序只支持number类型
scroll-top设置竖向滚动条位置支持支付宝小程序只支持number类型
scroll-left设置横向滚动条位置支持支付宝小程序只支持number类型
scroll-into-view值应为某子元素id（id不能以数字开头）。设置哪个方向可滚动，则在哪个方向滚动到该元素完全支持支付宝小程序完全支持该属性
scroll-with-animation在设置滚动条位置时使用动画过渡完全支持支付宝小程序完全支持该属性
enable-back-to-top当点击 iOS 顶部状态栏或者双击安卓标题栏时，滚动条返回顶部，只支持竖向完全支持支付宝小程序完全支持该属性
bindscrolltoupper滚动到顶部/左边时触发完全支持完全支持
bindscrolltolower滚动到底部/右边时触发完全支持完全支持
bindscroll滚动时触发，event.detail = {scrollLeft, scrollTop, scrollHeight, scrollWidth, deltaX, deltaY}完全支持完全支持

属性名说明是否支持备注
current-item-idmissing不支持暂不支持

indicator-color指示点颜色完全支持支付宝小程序完全支持该属性
indicator-active-color当前选中的指示点颜色完全支持支付宝小程序完全支持该属性
autoplay是否自动切换完全支持支付宝小程序完全支持该属性
current当前所在滑块的 index完全支持支付宝小程序完全支持该属性
interval自动切换时间间隔完全支持支付宝小程序完全支持该属性
duration滑动动画时长完全支持支付宝小程序完全支持该属性
circular是否启用无限滑动完全支持支付宝小程序完全支持该属性
vertical滑动方向是否为纵向完全支持支付宝小程序完全支持该属性
previous-margin前边距，可用于露出前一项的一小部分，接受 px 和 rpx 值完全支持支付宝小程序完全支持该属性
