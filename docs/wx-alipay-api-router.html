路由

switchTab

函数名说明微信小程序支付宝小程序是否支持
switchTab跳转到 tabBar 页面，并关闭其他所有非 tabBar 页面docdoc完全支持

reLaunch

函数名说明微信小程序支付宝小程序是否支持
reLaunch关闭所有页面，打开到应用内的某个页面docdoc支持

redirectTo

函数名说明微信小程序支付宝小程序是否支持
redirectTo关闭当前页面，跳转到应用内的某个页面。但是不允许跳转到 tabbar 页面docdoc完全支持

navigateTo

函数名说明微信小程序支付宝小程序是否支持
navigateTo保留当前页面，跳转到应用内的某个页面。但是不能跳到 tabbar 页面。使用 wx.navigateBack 可以返回到原页面。小程序中页面栈最多十层。docdoc完全支持

navigateBack

函数名说明微信小程序支付宝小程序是否支持
navigateBack关闭当前页面，返回上一页面或多级页面docdoc完全支持
