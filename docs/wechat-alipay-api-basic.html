基础

canIUse

函数名说明微信小程序支付宝小程序是否支持
canIUse判断小程序的 API，回调，参数，组件等是否在当前版本可用docdoc完全支持

getSystemInfoSync

函数名说明微信小程序支付宝小程序是否支持
getSystemInfoSync获取系统信息同步版本docdoc支持

  • 返回值属性值缺失
差异属性说明差异类型
SDKVersion客户端基础库版本不支持该属性
benchmarkLevel仅 Android 小游戏 设备性能等级取值为：-2 或 0不支持该属性
albumAuthorized允许微信使用相册的开关 仅 iOS 有效不支持该属性
cameraAuthorized允许微信使用摄像头的开关不支持该属性
locationAuthorized允许微信使用定位的开关不支持该属性
microphoneAuthorized允许微信使用麦克风的开关不支持该属性
notificationAuthorized允许微信通知的开关不支持该属性
notificationAlertAuthorized允许微信通知带有提醒的开关 仅 iOS 有效不支持该属性
notificationBadgeAuthorized允许微信通知带有标记的开关 仅 iOS 有效不支持该属性
notificationSoundAuthorized允许微信通知带有声音的开关 仅 iOS 有效不支持该属性
bluetoothEnabled蓝牙的系统开关不支持该属性
locationEnabled地理位置的系统开关不支持该属性
wifiEnabledWi-Fi 的系统开关不支持该属性
