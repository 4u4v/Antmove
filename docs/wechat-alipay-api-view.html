界面

showToast

函数名说明微信小程序支付宝小程序是否支持
showToast显示消息提示框docdoc支持

  • 入参参数差异/缺失
差异参数说明差异类型
title提示的内容, wx: title, alipay: content命名及格式不同
icon图标, wx: icon, alipay: type命名及格式不同
image自定义图标的本地路径，image 的优先级高于 icon不支持该属性
mask是否显示透明蒙层，防止触摸穿透不支持该属性
icon的合法值显示加载图标，此时 title 文本最多显示 7 个汉字长度不支持该属性

showModal

函数名说明微信小程序支付宝小程序是否支持
showModal显示模态对话框docdoc支持

  • 入参参数名称差异/参数缺失/返回值名称差异
差异参数说明差异类型
confirmText确认按钮的文字，最多 4 个字符, wx: confirmText, alipay: confirmButtonText命名及格式不同
cancelText取消按钮的文字，最多 4 个字符, wx: cancelText, alipay: cancelButtonText命名及格式不同
showCancel是否显示取消按钮不支持该属性
cancelColor取消按钮的文字颜色，必须是 16 进制格式的颜色字符串不支持该属性
confirmColor确认按钮的文字颜色，必须是 16 进制格式的颜色字符串不支持该属性
差异属性说明差异类型
confirmwx: 为true时，表示用户点击了确定按钮, alipay: 点击confirm返回true，点击cancel返回false命名及格式不同
cancelwx: 为 true 时，表示用户点击了取消（用于 Android 系统区分点击蒙层关闭还是点击取消按钮关闭）命名及格式不同
