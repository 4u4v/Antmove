  • offAudioInterruptionEnd - 取消监听音频中断结束事件
  • offAppShow - 取消监听小程序切前台事件
  • offAppHide - 取消监听小程序切后台事件

  • onGetWifiList - 监听获取到 Wi-Fi 列表数据事件

  • closeBLEConnection - 断开与低功耗蓝牙设备的连接。

  • getBatteryInfo - 获取设备电量。同步 API wx.getBatteryInfoSync 在 iOS 上不可用。

  • startHCE - 初始化 NFC 模块。
  • sendHCEMessage - 发送 NFC 消息。仅在安卓系统下有效。
  • onHCEMessage - 监听接收 NFC 设备消息事件
  • getHCEState - 判断当前设备是否支持 HCE 能力。

  • startCompass - 开始监听罗盘数据

  • startDeviceMotionListening - 开始监听设备方向的变化
  • stopDeviceMotionListening - 停止监听设备方向的变化
  • startGyroscope - 开始监听陀螺仪数据
  • onMemoryWarning - 监听内存不足告警事件
