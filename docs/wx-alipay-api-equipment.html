onBLEConnectionStateChange监听低功耗蓝牙连接状态的改变事件。包括开发者主动连接或断开连接，设备丢失，连接异常断开等等docdoc支持

onBLECharacteristicValueChange

函数名说明微信小程序支付宝小程序是否支持
onBLECharacteristicValueChange监听低功耗蓝牙设备的特征值变化事件。必须先启用 notifyBLECharacteristicValueChange 接口才能接收到设备推送的 notification。docdoc支持

  • 返回值类型不同
不支持的属性说明
value蓝牙设备特征值的value支持数据类型不同,wx: ArrayBuffer, alipay: Hex String

notifyBLECharacteristicValueChange

函数名说明微信小程序支付宝小程序是否支持
notifyBLECharacteristicValueChange监听低功耗蓝牙设备的特征值变化事件。必须先启用 notifyBLECharacteristicValueChange 接口才能接收到设备推送的 notification。docdoc支持

getBLEDeviceCharacteristics

函数名说明微信小程序支付宝小程序是否支持
getBLEDeviceCharacteristics获取蓝牙设备某个服务中所有特征值(characteristic)。docdoc支持
