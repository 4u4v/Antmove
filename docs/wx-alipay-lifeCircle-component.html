组件

组件

created

函数名说明微信小程序支付宝小程序是否支持
created在组件实例刚刚被创建时执行docdoc完全支持

attached

函数名说明微信小程序支付宝小程序是否支持
attached在组件实例进入页面节点树时执行docdoc完全支持

ready

函数名说明微信小程序支付宝小程序是否支持
ready在组件在视图层布局完成后执行docdoc完全支持

detached

函数名说明微信小程序支付宝小程序是否支持
detached在组件实例被从页面节点树移除时执行docdoc完全支持

moved

函数名说明微信小程序支付宝小程序是否支持
moved在组件实例被移动到节点树另一个位置时执行docdoc不支持

error

函数名说明微信小程序支付宝小程序是否支持
error每当组件方法抛出错误时执行docdoc不支持

show

函数名说明微信小程序支付宝小程序是否支持
show组件所在的页面被展示时执行docdoc不支持

hide

函数名说明微信小程序支付宝小程序是否支持
hide组件所在的页面被隐藏时执行docdoc不支持

resize

函数名说明微信小程序支付宝小程序是否支持
resize组件所在的页面尺寸变化时执行docdoc不支持
