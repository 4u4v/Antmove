差异属性说明是否支持返回值差异备注
navigationBarBackgroundColor导航栏背景颜色，如 #000000完全支持titleBarColor
navigationBarTextStyle导航栏标题颜色，仅支持 black / white不支持暂不支持

navigationStyle导航栏样式不支持暂不支持
backgroundColor窗口的背景色完全支持
backgroundTextStyle下拉 loading 的样式，仅支持 dark / light不支持暂不支持
backgroundColorTop顶部窗口的背景色，仅 iOS 支持不支持暂不支持
backgroundColorBottom底部窗口的背景色，仅 iOS 支持不支持暂不支持
enablePullDownRefresh是否开启全局的下拉刷新完全支持pullRefresh
onReachBottomDistance页面上拉触底事件触发时距页面底部距离，单位为 px不支持暂不支持
pageOrientation屏幕旋转设置，支持 auto / portrait / landscape,详见 响应显示区域变化不支持暂不支持
